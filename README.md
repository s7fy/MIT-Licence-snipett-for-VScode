# Visual Studio Code MIT Licence snippets

## Set up
ファイル -> ユーザ設定 -> ユーザースニペットの構成 
-> 新しいグローバルスニペット -> MIT_Licence -> 同封されたMITLicence.txtからコードを貼り付け -> 上書き保存

## Usage
Just type to "MITLicence" for row in editor where you like.
